Virtual Machies Creation Failure via Azure Portal
Summary of impact: Between 17:40 and 20:20 UTC on 05 Mar 2018, a subset of customers in all regions may have received failure notifications when performing Virtual Machine create job via Microsoft Azure Portal. Virtual Machines creation using PowerShell was not affected during this time. Preliminary root cause: Engineers determined that a recent Marketplace deployment was preventing requests from completing. Mitigation: Engineers rolled back the recent deployment task to mitigate the issue. Next steps: Engineers will continue to investigate to establish the full root cause and prevent future occurrences.
