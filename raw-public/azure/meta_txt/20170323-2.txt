Azure Active Directory 
Summary of impact: Between 22:00 UTC on 22 Mar 2017 and 00:30 UTC on 23 Mar 2017, Azure Active Directory encountered an issue that affected service to service authentication that impacted multiple Azure services. Azure Resource Manager, Logic Apps, Azure Monitor, Azure Data Lake Analytics, and Azure Data Lake Store customers may have experienced intermittent service management issues as downstream impact. AAD customers would not have seen impact. Preliminary root cause: Engineers identified a recent configuration change as the potential root cause. Mitigation: Engineers rolled back the configuration change to mitigate the issue.
